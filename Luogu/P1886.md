# P1886 滑动窗口 /【模板】单调队列

## 题目描述

有一个长为 $n$ 的序列 $a$，以及一个大小为 $k$ 的窗口。窗口从左边开始向右滑动，每次滑动一个单位，求出每次滑动后窗口中的最小值和最大值。

## 输入格式

第一行两个正整数 $n,k$；
第二行 $n$ 个整数，表示序列 $a$。

## 输出格式

第一行每次窗口滑动的最小值；
第二行每次窗口滑动的最大值。

## 数据范围

$1 \le k \le n \le 10^6$，$a_i \in [-2^{31}, 2^{31})$

---

## 思路

### 暴力做法

每次滑动窗口都遍历 $k$ 个元素找最值，时间复杂度 $O(nk)$，$n=10^6$ 时完全不可行。

### 单调队列优化（$O(n)$）

核心思想：**维护一个双端队列，其中存的是下标，且对应的值是单调的**。

#### 求最小值

队列单调**递增**，队头就是当前窗口的最小值。

每次处理新元素 $a[i]$：
1. **去尾**：把队尾 $\ge a[i]$ 的下标全部弹出（这些值在 $a[i]$ 存在期间永远不可能是最小值了）
2. **入队**：将 $i$ 压入队尾
3. **去头**：如果队头下标 $\le i-k$ 说明已离开窗口，弹出队头
4. **记录**：队头对应的值就是当前窗口的最小值

#### 求最大值

对称地，队列单调**递减**，队头就是最大值。其他操作完全对称。

### 复杂度

- 每个下标最多入队一次、出队一次，均摊 $O(1)$
- 总时间复杂度 **$O(n)$**
- 空间复杂度 $O(k)$（队列最多同时存 $k$ 个下标）

## 代码

```cpp
#include<bits/stdc++.h>
using i64 = long long;

void solve() {
    int n, k;
    std::cin >> n >> k;

    std::vector<int> a(n);
    for (int i = 0; i < n; i++) {
        std::cin >> a[i];
    }

    std::deque<int> dq;
    std::vector<int> mn, mx;

    for (int i = 0; i < n; i++) {
        // 去尾：保持队列单调递增（队头最小）
        while (!dq.empty() && a[dq.back()] >= a[i]) {
            dq.pop_back();
        }
        dq.push_back(i);

        // 去头：移除超出窗口范围的
        if (dq.front() <= i - k) {
            dq.pop_front();
        }

        // 窗口形成后开始记录
        if (i >= k - 1) {
            mn.push_back(a[dq.front()]);
        }
    }

    dq.clear();

    for (int i = 0; i < n; i++) {
        // 去尾：保持队列单调递减（队头最大）
        while (!dq.empty() && a[dq.back()] <= a[i]) {
            dq.pop_back();
        }
        dq.push_back(i);

        if (dq.front() <= i - k) {
            dq.pop_front();
        }

        if (i >= k - 1) {
            mx.push_back(a[dq.front()]);
        }
    }

    for (int i = 0; i < (int)mn.size(); i++) {
        std::cout << mn[i] << " \n"[i == (int)mn.size() - 1];
    }
    for (int i = 0; i < (int)mx.size(); i++) {
        std::cout << mx[i] << " \n"[i == (int)mx.size() - 1];
    }
}

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int tt = 1;
    // std::cin >> tt;

    while (tt--) {
        solve();
    }

    return 0;
}
```

## 补充：合并一次遍历的写法

最小值、最大值可以在同一个循环里用两个队列分别维护，也可以拆开写（更清晰）。上面的代码为了可读性拆成了两个循环。

如果追求常数优化，可以合并成一个循环，开两个 deque 同时维护即可：

```cpp
for (int i = 0; i < n; i++) {
    while (!dmn.empty() && a[dmn.back()] >= a[i]) dmn.pop_back();
    while (!dmx.empty() && a[dmx.back()] <= a[i]) dmx.pop_back();
    dmn.push_back(i);
    dmx.push_back(i);
    if (dmn.front() <= i - k) dmn.pop_front();
    if (dmx.front() <= i - k) dmx.pop_front();
    if (i >= k - 1) {
        std::cout << a[dmn.front()] << " \n"[i == n - 1];
        // 存起来最后一起输出也行
    }
}
```

但输出格式要求最小值一行、最大值一行，分开写两个循环反而更自然。
