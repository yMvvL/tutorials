# P3406 海底高铁
### 问题转化

铁路有 $N$ 个城市、$N-1$ 段铁路。Uim 会按顺序访问 $M$ 个城市，经过一段铁路多次。

每段铁路有两种付费方式：
- **买纸质票**：每次 $A_i$ 元，无固定成本
- **买 IC 卡**：工本费 $C_i$，之后每次 $B_i$ 元（$B_i < A_i$）

目标：最小化总花费。

**关键观察**：每段铁路独立决策。只需要知道每段铁路被经过的次数 $cnt_i$，则该段最小花费为：
$$
\min(A_i \times cnt_i,\ C_i + B_i \times cnt_i)
$$

### 如何统计经过次数

给定访问序列 $P_1, P_2, \dots, P_M$，相邻两个城市 $P_j, P_{j+1}$ 之间，所有中间的铁路段都会经过一次。

例如 $P_j = 3, P_{j+1} = 1$，则经过铁路段 $1, 2$（从 3→2→1）。

直接对每对相邻城市遍历区间是 $O(NM)$ 的，会超时（$N,M \le 10^5$）。

用**差分数组**优化：对于每次行程 $(l, r)$（$l \le r$），给区间 $[l, r-1]$ 加 1。转化为：
$$
\text{diff}[l] \text{++},\quad \text{diff}[r] \text{--}
$$
最后前缀和还原 $cnt_i$。时间复杂度 $O(N+M)$。

### 算法流程

1. 读入 $N, M$ 和访问序列 $P$
2. 差分数组统计每段铁路经过次数
3. 对每段铁路 $i$，计算 $\min(A_i \times cnt_i,\ C_i + B_i \times cnt_i)$ 累加
4. 输出答案

### 复杂度分析

- **时间复杂度**：$O(N + M)$
- **空间复杂度**：$O(N)$

## 代码

```cpp
#include <bits/stdc++.h>
using i64 = long long;

void solve() {
    int n, m;
    std::cin >> n >> m;

    std::vector<int> p(m);
    for (int i = 0; i < m; i++) {
        std::cin >> p[i];
    }

    std::vector<int> a(n), b(n), c(n);
    for (int i = 1; i < n; i++) {
        std::cin >> a[i] >> b[i] >> c[i];
    }

    std::vector<i64> d(n + 2, 0);
    for (int i = 0; i < m - 1; i++) {
        int l = p[i];
        int r = p[i + 1];
        if (l > r) {
            std::swap(l, r);
        }
        d[l]++;
        d[r]--;
    }

    i64 ans = 0;
    i64 s = 0;
    for (int i = 1; i < n; i++) {
        s += d[i];
        ans += std::min(a[i] * s, c[i] + b[i] * s);
    }

    std::cout << ans << "\n";
}

int main() {
    std::ios::sync_with_stdio(false);
    std::cin.tie(nullptr);

    int tt = 1;
    // std::cin >> tt;

    while (tt--) {
        solve();
    }

    return 0;
}
```
